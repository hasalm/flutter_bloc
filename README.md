# flutter_bloc
Discovering The fundamentals of Flutter Bloc State Management with Simple Application- Inspired By: Rivaan Ranawat Developer.

00- Flutter Basic Counter Application With Cubit: Extending the Flutter Basic Counter Application with Bloc state management using the Cubit Interface.

01- Flutter Basic Counter Application With Bloc : Extending the Flutter Basic Counter Application with Bloc state management using the Bloc Interface.

02- Basic Login Form Application With Bloc      : Making a Small Login form in order to explore data handling and validation using Bloc state Management.

03- TodoList Application using Cubit            : Making a small TodoList Application  with Bloc state management using Cubit.

04- Weather Forecast Application With Bloc      : A small Weather Forecast Application in which we explore the Bloc Architecutre in depth using Bloc State Management.

# Usage

1- Download the Repository.

2- Replace the lib folder with your lib folder in your flutter project.

3- Replace the pubspec.yaml file also or just download the dependencies.

4- Add the assets file and configure it inside of your pubspec.yaml file (already configured and included)

5- Enjoy Exploring(Bloc state management with Bloc/Cubit Interfaces and Bloc Architecture).
